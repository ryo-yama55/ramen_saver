## 🎯 デザインコンセプト

### エンターテインメント性 × 親しみやすさ

**✅ 目指すもの**:
- 楽しい、ポジティブな印象
- 達成感を感じられるデザイン
- ゲーム的な要素
- シンプルで分かりやすい

**❌ 避けるもの**:
- お堅い、説教臭い雰囲気
- 過度に派手、子供っぽすぎる

## ❓ 要確認事項

デザインブリーフの「要確認事項」セクションに、以下の項目が記載されています：

1. データ永続化戦略
2. 認証の必要性
3. デスクトップ版のレイアウト
4. 「今月」の定義
5. 1日に複数回記録可能か
6. 通貨表示フォーマット
7. 応援メッセージのバリエーション
8. ウェルカム画面の表示頻度
9. タイムゾーン処理
10. アクセシビリティ要件

各項目の詳細は[デザインブリーフ](./design_brief.md#要確認事項設計判断が必要な項目)を参照してください。
